# 小手机系统 PRD（产品需求文档）

## 一、产品定位

**小手机系统**是一款纯前端虚拟手机系统，部署在静态托管（InfinityFree 等）即可运行。

- **形态**：单页 Web 应用，桌面/移动浏览器自适应，max-width 600px 居中
- **核心目标**：所有 APP 真实联动，事件统一汇入"消息中心"，AI 可读取并提及
- **高自由度**：壁纸/主题/图标顺序/Dock/角色/世界书/商店商品/钱包余额等全部可自定义
- **专业体验**：每个 APP 都像真实 APP 一样完整好用，不是"能用就行"的儿童玩具

## 二、用户与场景

- **目标用户**：喜欢陪伴型 AI、想要一个可深度自定义的虚拟伴侣手机的人
- **核心场景**：
  1. 与 AI 角色日常聊天（气泡 / 对话双模式）
  2. 管理多个 AI 角色人设，切换不同伴侣
  3. 通过朋友圈、礼物、纪念日、游戏等加深情感联动
  4. 自定义系统外观和功能，打造专属"小手机"

## 三、风格定位

**Soft Cozy Minimal**（温柔软萌极简风）：
- 棉花糖甜，不是硬糖甜
- 干净、精致、轻盈、有温度、不塑料
- 默认主题：奶黄色系（honey）
- 字体：PingFang SC / HarmonyOS Sans SC
- 形状：大圆角（16-28px），禁止直角
- 图标：线条风 stroke-width 1.5px，禁止填充和 emoji

## 四、核心功能模块

### 4.1 桌面系统
- 壁纸：默认奶黄渐变 / 本地上传 base64 / URL 图床
- 顶部状态栏：可爱状态胶囊，9 种线条风 SVG 图标轮换（爱心/猫脸/熊/爪印/笑脸/音符/星星/月亮/大爪印）
- 桌面小组件 4 个：时间 / 天气 / 今日提示 / 黑胶唱片（可点击旋转）
- APP 图标网格：4 列，默认 13 个 APP，长按可编辑（拖拽/删除/隐藏）
- 底部 Dock：4 个图标，毛玻璃背景，默认 消息/设置/角色/世界书
- 页面指示器：2 圆点
- 锁屏：4 位数字密码（默认 0326），错误提示"嘿嘿，不对哦"，壁纸/头像/文案可自定义
- 加载动画："小手机正在醒来..." + 呼吸动画

### 4.2 消息中心（对齐微信/QQ）
- 消息列表：头像/名字/最后消息/时间/未读红点，置顶，左滑操作，搜索，新建
- 聊天界面：文字/图片/语音气泡，左右区分，长按操作（复制/引用/转发/撤回/删除/收藏）
- 输入工具栏：文字/表情/图片/语音，打字状态显示，消息时间分组，滚动到底部按钮，草稿自动保存
- 两种聊天模式：气泡模式 / 对话模式，可自由切换
- 聊天设置：对方资料页 / 免打扰 / 置顶 / 记录导出清空 / 单独背景
- 流式输出（fetch + ReadableStream）、防抖提交、虚拟滚动、dvh 适配软键盘

### 4.3 朋友圈（对齐 ex 朋友圈）
- 时间线倒序展示
- AI 自动发圈 / 用户手动发圈，文字+图片（最多9张），可见范围
- 点赞 / 评论 / 回复 / 删除
- AI 联动：根据人设自动发圈、点赞评论用户、聊天中可提及

### 4.4 记仇本
- AI 不开心情绪自动写入
- 记录触发事件/时间/原因
- 手动管理：编辑/删除/标记原谅
- 影响情绪与回复，用户可道歉被原谅

### 4.5 钱包与商店
- 钱包：用户余额自定义 / AI 初始 5k / 交易记录 / 双向转账
- 商店：商品分类 / 用户自定义商品 / 购买流程 / 赠送 AI
- 联动：购买触发事件，AI 提及礼物，余额变化通知

### 4.6 游戏中心
- 真心话大冒险 / 谁是卧底 / 骗子酒馆 / 塔罗牌占卜 / 其他创意
- 游戏结果可分享朋友圈，触发事件，AI 聊天提及

### 4.7 音乐播放器（对齐网易云）
- 音乐库：上传 MP3/WAV/FLAC，封面/歌名/歌手/时长
- 播放：上下首/进度条/音量/顺序/随机/单曲循环
- 播放界面：封面/歌词/动画
- 歌单管理
- AI 推荐 / 分享朋友圈 / 聊天提及

### 4.8 设置中心
- 个性化：主题（honey/pink/sky）/ 壁纸 / 字号 / 桌面布局 / 图标自定义 / Dock / 各 APP 背景
- AI 与接口：API 地址 / Key / 模型 / 角色管理 / 说话风格 / 思维链开关
- 通知：总开关 / 分 APP / 免打扰时段 / 角标
- 锁屏与安全：密码 / 壁纸 / 头像 / 文案
- 聊天：气泡样式 / 字号 / 背景
- 数据管理：导出 JSON / 导入 / 清空 / 重置
- 关于：版本号 / 系统名字（可自定义）

### 4.9 角色管理
- 角色卡片：头像/名字/简介/人设/说话方式/背景/世界书关联/独立记忆
- 创建/编辑/删除/切换当前角色

### 4.10 世界书
- 条目：关键词/触发词/内容/优先级/启用
- 管理：CRUD / 分类 / 搜索 / 导入导出
- 联动：聊天检测关键词，自动插入 prompt

### 4.11 记忆系统
- 记忆条目：内容/时间/角色/类型（对话/事件/喜好/事实）/重要度
- 管理：查看/编辑/删除/筛选/导出/清空
- **记忆隔离**：每个角色独立记忆，A 不知 B

### 4.12 备忘录
- 列表/搜索/CRUD/标记完成/提醒时间
- AI 读取并在聊天提及

### 4.13 周年纪念
- 列表/倒计时/CRUD/提醒（当天/提前1/3/7天）
- 当天 AI 主动提及，提醒写入事件中心

## 五、AI 接口规范

- OpenAI 兼容格式
- 自定义接口地址 / Key / 模型
- 流式输出（stream:true）
- 友好错误处理（网络/超时/API 错误均不显示技术细节）

## 六、数据存储

- IndexedDB 持久化
- 提供 JSON 导出/导入
- 图片优先 URL 模式，base64 占空间
- 定期清理过期数据

## 七、技术约束

- **不用 ES Module**，用 IIFE + 全局对象 `window.Phone`，避免本地双击白屏
- 单文件不超过 800 行
- 业务/界面/样式/数据/AI 逻辑分离
- APP 注册表 + 事件中心 + 统一数据源
- APP 间不直接互调，全部走事件中心

## 八、自由度原则

**所有可自定义处必须可自定义**，禁止写死：
系统名/壁纸/主题/桌面图标顺序/Dock/默认角色/默认称呼/通知文案/AI 说话方式/锁屏配置/各 APP 背景/商店商品/钱包余额/游戏内容/纪念日/备忘录/世界书/角色人设。

## 九、文案与语气

- 所有界面文案可爱（"小手机正在醒来"、"嘿嘿，不对哦"）
- 禁止干巴巴纯技术文案
- 错误提示友好，不显示技术细节

## 十、PWA 与离线

- manifest.json + service-worker.js
- 首次打开后缓存静态文件，秒开 + 断网可用
- 纯前端不做关闭后主动推送（后续考虑）

## 十一、验收标准

1. 桌面加载动画 → 锁屏 → 桌面全流程通畅
2. 13 个 APP 全部可进入且功能完整（不是假按钮）
3. APP 事件真实写入事件中心，可在消息中心查看
4. AI 聊天流式输出正常，能读取并提及各 APP 数据
5. 三套主题切换无破图
6. 移动端/桌面端比例一致，无错位
7. 数据可导出/导入
8. 风格统一为 Soft Cozy Minimal，无硬编码色值/灰色阴影/纯黑
